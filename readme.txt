==Description==

This plugin will add a author box functionality with author bio and social link.
The author box will be added bellow the content of single post.

==Supported social profile==
*Facebook
*Twitter
*Linkedin
*Pinterest
*Dribbble
*Instagram
*Skype
*Youtube
*Vimeo

==Installation==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `focus-author-bio` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3.You have done... Now check in single post
